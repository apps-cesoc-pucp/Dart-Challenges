# CHALLENGE 1: A VERY BIG SUM

## Problem
Calculate and print the sum of the elements in a list.

## Input Format
A list with numbers

## Output Format
The sum of the elements in the list

## Constraints

```
0 <= ar[i] <= 10^10
```

## Example 
### Input
```
[1000, 2000, 3000, 4000, 5000]
```
### Output
```
15000
```

## Knowledge
+ Work with lists
+ How to use for loops
